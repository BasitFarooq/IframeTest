Attributes of an Advertisement
  • Order Date: When the banner was ordered.
  • Exposures: Number of exposures this banner has been presented to end user
  • Start Date: The first day the advertisement will be active
  • End Date: The expiration date when the banner will be deactivated (we will contact you via email when this date is approaching)
  • Price: The total investment for this banner
  • Status: If the banner is still active or deactivated
